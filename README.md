# 📦 Need a Part? - Parts Leaderboard System

الیکٹرانکس پرزہ جات کی ڈیمانڈ ریکارڈنگ، پوائنٹس، اور Buying Order سسٹم۔

## 🎯 فیچرز
- ✅ صارف رجسٹریشن (اپنی فیلڈ خود لکھے — UPS, Solar, LED TV وغیرہ)
- ✅ ہر پرزے کے ساتھ ڈیوائس آئٹم کا ذکر
- ✅ 5 پوائنٹس عام، 10 پوائنٹس مشترکہ پرزے پر
- ✅ ٹاپ 10 لیڈر بورڈ (3 ماہ)
- ✅ ایڈمن: مشترکہ پرزوں کی Buying Order

## 🚀 انسٹالیشن
1. فائلیں `www` فولڈر میں ڈالیں
2. phpMyAdmin میں `parts_leaderboard` بنائیں
3. `database.sql` امپورٹ کریں
4. `http://localhost/parts-leaderboard/signup.php` کھولیں

## 📄 فائلیں
| فائل | کام |
|---|---|
| signup.php | رجسٹریشن |
| index.php | پرزہ درج کریں |
| leaderboard.php | ٹاپ 10 |
| admin.php | Buying Order |
| database.sql | ڈیٹا بیس |
